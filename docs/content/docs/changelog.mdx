## 1.0.0

Initial release.

- Self-hosted push notifications via Expo Push Service
- Auto-notify on post publish
- Manual send with live preview, character counters, and placeholder quick-fill
- Notification scheduling via WP-Cron
- Device management with search, edit, dev flag, and bulk operations
- Device groups for targeted notifications
- Rich notifications with featured images
- Dashboard with stats, charts, and recent activity
- CSV import/export for devices
- Template system with placeholders
- Post editor meta box for per-post control
- Notification history with status tracking
- Dev Mode for testing
- REST API for device registration, send, stats, and import/export
- Clean uninstall of all tables, options, and capabilities
- Tailwind CSS admin UI (pre-compiled, no CDN)
- PHPUnit test suite (169 tests, 263 assertions)
- GitHub Actions CI/CD (test, build, release)
