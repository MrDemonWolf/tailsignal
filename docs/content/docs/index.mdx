## What is TailSignal?

TailSignal is a self-hosted WordPress plugin that sends push notifications to mobile devices via the **Expo Push Service**. It replaces third-party services like OneSignal, keeping all data on your own server.

- **Self-hosted** — your data never leaves your server
- **Expo Push API** — works with any Expo/React Native mobile app
- **Auto-notify** — automatically send when posts or portfolio items are published
- **Rich notifications** — include images, custom data, and deep linking
- **Device management** — groups, dev mode, CSV import/export
- **Scheduling** — send notifications at a future time via WP-Cron
